What is Django?

Django Reinhardt

* 23.1.1910✝ 16.5.1953

General

  • Web Application Framework
  • Written using the Python programming language
  • Open Source Software (BSD Licence)
  • Django Software Foundation (DSF)
  • Extensive Documentation
  • Huge, friendly Community

Philosophy

  • Rapid Development
  • Loosely Coupled
  • Reusable Applications
  • Don't Repeat Yourself (DRY)
Every piece of knowledge must have a single, unambiguous, authoritative representation within a system.

  • South - database migrations
  • django-debug-toolbar - beautiful debugging toolbar, extensible with many extensions
  • django-devserver - drop in replacement for the built-in runserver command
  • django-braces - a few nice-to-have class helpers for class-based views
  • django-crispy-forms - reusable layouts without writing HTML in templates
  • django-parsley - client side form validation using Parsley.js
  • django-haystack - modular search app supporting Solr, Elasticsearch, Whoosh and Xapian
  • django-celery - Django integration for the asynchronous task queue/job queue Celery
  • django-tastypie - REST framework for Django
  • djangorestframework - REST framework for Django
  • factory_boy - fixture generator
  • httpretty - mocks HTTP requests
  • django-jenkins - easy integration with Jenkins
  • gunicorn - Python WSGI HTTP server for Unix
  • django_compressor - Compresses JavaScript or CSS into a single cached file
  • django-secure - utilities to make your Django site more secure
